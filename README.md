# City Slicker

## Getting Started

### Dependencies

- Ruby 3.2.2
- Rails 7.1.2
- Node 19.4.0
- NPM 9.6.5

### Local setup

1. clone repo and cd into it
2. `bundle install`
3. `npm install`
4. `yarn install`
5. `bundle exec rails db:setup`
5. `bundle exec rails db:migrate`
5. `bundle exec rails db:seed`
6. `bundle exec rails s`
7. open [http://localhost:3000/](http://localhost:3000/)
