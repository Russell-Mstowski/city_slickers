# City Slicker

## Getting Started

### Dependencies

- ruby 3.2.2
- rails 7.1.2
- node 19.4.0
- npm 9.6.5

### Local setup

1. clone repo and `cd` into it
2. `bundle`
3. `npm i`
4. `yarn install`
5. `bundle exec rails db:setup`
6. `bundle exec rails db:migrate`
7. `bundle exec rails db:seed`
8. `bundle exec rails s`
9. open [http://localhost:3000/](http://localhost:3000/)
